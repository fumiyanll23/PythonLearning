# D - Fibonacci Sequence

## 問題文

正整数 N が与えられる．
Fibonacci 数列の第 N 項を返す関数 `fibonacci` を作成せよ．

## 制約

- N は正整数である．

## 入力

入力は以下の形式で標準入力から与えられる．

```
N
```

## 出力

Fibonacci 数列の第 N 項を出力せよ．

## 入力例1

```
1
```

## 出力例1

```
1
```

## 入力例2

```
2
```

## 出力例2

```
1
```

## 入力例3

```
30
```

## 出力例3

```
832040
```
